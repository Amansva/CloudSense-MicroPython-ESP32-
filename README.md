# CloudSense-MicroPython-ESP32-
* IoT monitoring system developed using **MicroPython** on **ESP32**
* Sensor data is transmitted to **ThingSpeak Cloud** using **Wi-Fi**
* Enables real-time remote monitoring and data visualization

## Hardware
* ESP32
* Sensors (temperature, humidity, soil moisture, etc.)

## Software
* MicroPython
* Thonny / uPyCraft
* ThingSpeak Cloud (HTTP over Wi-Fi)

## How to Run
1. Flash MicroPython on ESP32
2. Configure Wi-Fi credentials and ThingSpeak API key
3. Upload code and view data on ThingSpeak dashboards

## Applications
* Environmental monitoring
* Smart agriculture
* Academic mini-projects
